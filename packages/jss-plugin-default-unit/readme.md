> JSS plugin that adds default custom unit to numeric values where needed

See our website [jss-plugin-default-unit](https://cssinjs.org/jss-plugin-default-unit?v=v10.0.0-alpha.24) for more information.

## Install

Using npm:

```sh
npm install jss-plugin-default-unit
```

or using yarn:

```sh
yarn add jss-plugin-default-unit
```
