# jss-plugin-vendor-prefixer

> JSS plugin that handles vendor prefixes in the browser

See our website [jss-plugin-vendor-prefixer](https://cssinjs.org/jss-plugin-vendor-prefixer?v=v10.0.0-alpha.9) for more information.

## Install

Using npm:

```sh
npm install jss-plugin-vendor-prefixer
```

or using yarn:

```sh
yarn add jss-plugin-vendor-prefixer
```
